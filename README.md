**Frontend**
React native expo

### Run in local

step 1

``git clone https://github.com/ArjunChauhan23/Task.git``

step-2

```
create .env file Pate
MONGO_PASSWORD=YOUR_PASSWORD
JWT_KEY=YOUR_JWT_KEY
BASEURL=http//localhost:5000
```

step - 3

````
cd App 
npm install && npm android
cd backend 
npm install && nodemon 
````
